static

发表于 2019-02-22

普通的成员属性和方法属于实例对象
而静态成员 (属性和方法) 属于类或者构造函数 可以被所有实例对象共享
静态方法 多用于创建一个应用程序的实例

对于不变的和需要被共享数据 设置静态属性

把不依赖于对象或者用来操作对象的方法 设为静态方法

通常面向对象中,使用 static 来声明静态方法

es6 class 语法中支持 ststic 定义静态方法 但是静态属性 还要早类上定义
